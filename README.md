## Stack technique

- React 18 + Vite
- Bootstrap 5 + SCSS
- Déploiement automatique via GitHub Actions → GitHub Pages
- Formulaire de contact via EmailJS (sans backend)

## Lancer en local

```bash
git clone https://github.com/sayouba2/<nom-du-repo>.git
cd <nom-du-repo>
npm install
npm run dev
```

## Déploiement

Le déploiement sur GitHub Pages est automatique à chaque push sur `main` via le workflow `.github/workflows/deploy.yml`.

Pour activer GitHub Pages sur ton repo :
1. Va dans **Settings → Pages**
2. Source : branche `gh-pages`, dossier `/ (root)`
